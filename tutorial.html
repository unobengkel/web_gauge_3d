Tutorial Web HTML: Setup Gauge Widget 3D

Panduan langkah demi langkah untuk mengintegrasikan Widget Gauge 3D (Three.js) ke dalam halaman web HTML Anda.

Langkah 1: Include Dependencies (JS & CSS)

Anda memerlukan library Three.js untuk merender 3D, beserta CSS dan JS bawaan dari widget ini.

Tambahkan kode berikut di dalam tag <head> HTML Anda:

<!-- Three.js (wajib) -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/three.js/r128/three.min.js"></script>

<!-- CSS Widget Gauge -->
<link rel="stylesheet" href="asset/widgetgauge/gauge.css">

<!-- JS Widget Gauge -->
<script src="asset/widgetgauge/gauge.js"></script>

Langkah 2: Buat Wadah (Container) HTML

Siapkan elemen HTML untuk menampung widget 3D ini. Beri ukuran spesifik (width dan height).

<div id="gauge-container" style="width: 300px; height: 250px; position: relative;">
    <!-- Tempat teks display nilai (opsional) -->
    <div style="position: absolute; bottom: 10%; width: 100%; text-align: center;">
        <span id="nilai">0</span>°C
    </div>
</div>

Langkah 3: Inisialisasi Plugin dengan JavaScript

Panggil class GaugeWidget di dalam script Anda (sebaiknya setelah penataan DOM selesai / DOMContentLoaded).

<script>
document.addEventListener('DOMContentLoaded', () => {
    // Inisialisasi widget
    const gauge = new GaugeWidget('#gauge-container', {
        minValue: 0,
        maxValue: 100,
        initialValue: 0,
        // (Opsi) Ubah warna
        /* colors: {
            low: 0x10b981,
            mid: 0xf59e0b,
            high: 0xef4444
        } */
    });

    // Opsional: Hubungkan display teks ke gauge
    gauge.bindToDisplay(document.getElementById('nilai'));
    
    // Set nilai baru secara terprogram
    gauge.setValue(45); // menggunakan animasi
});
</script>

Langkah 4: Fitur Lanjutan

Anda dapat mengontrol widget secara lebih dinamis menggunakan API yang disediakan.

Hubungkan ke Slider

Bisa otomatis mengubah nilai widget mengikuti nilai slider input.

gauge.bindToSlider(document.getElementById('my-slider'));

Mode Simulasi

Dapat berjalan acak untuk melihat animasi warna.

gauge.startSimulation(2000); // ganti nilai setiap 2000ms
// gauge.stopSimulation(); // untuk berhenti

Merespon Perubahan via Event

gauge.onValueChange(function(newVal, oldVal) {
    console.log("Nilai berubah dari", oldVal, "ke", newVal);
});
